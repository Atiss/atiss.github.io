# nerelog

## First install node

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Docker build
```
docker build -f Dockerfile -t client .
```

### Docker run
```
docker run -it -p 4001:3000 client    
```
